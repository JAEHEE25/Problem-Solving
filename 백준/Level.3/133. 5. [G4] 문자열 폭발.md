# 5. [G4] 문자열 폭발

### [ 문제 정보 ]
> **백준**: https://www.acmicpc.net/problem/9935
> 
> **난이도**: 3
>
> **유형**: #자료구조


### [ 문제 풀이 ]
```Java
import java.util.*;
import java.lang.*;
import java.io.*;

class Main {
    static Stack<Character> stack;
    static String bomb;
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        stack = new Stack<>();
        String input = br.readLine();
        bomb = br.readLine();
        
        for (int i = 0; i < input.length(); i++) {
            char letter = input.charAt(i);
            stack.push(letter);
            if (letter == bomb.charAt(bomb.length() - 1)) {
                bomb(); // 마지막 글자일 때 폭발
            }
        }

        if (stack.isEmpty()) {
            System.out.println("FRULA");
        } else {
            StringBuilder answer = new StringBuilder();
            while (!stack.isEmpty()) {
                answer.append(stack.pop());
            }
            System.out.println(answer.reverse().toString());
        }
    }

    static void bomb() {
        StringBuilder sb = new StringBuilder();
        int b = bomb.length() - 1;
        while (!stack.isEmpty()) {
            char peek = stack.peek();
            if (b < 0 || peek != bomb.charAt(b)) {
                break;
            }
            sb.append(stack.pop());
            b--;
        }
        // 안 맞을 경우 다시 넣기
        if (b >= 0) {
            int i = sb.length() - 1;
            while (i >= 0) {
                stack.push(sb.charAt(i));
                i--;
            }
        }
    }
}
```
