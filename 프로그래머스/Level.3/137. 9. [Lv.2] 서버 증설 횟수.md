# 9. [Lv.2] 서버 증설 횟수

### [ 문제 정보 ]
> **프로그래머스**: https://school.programmers.co.kr/learn/courses/30/lessons/389479#
> 
> **난이도**: 3
>
> **유형**: #구현


### [ 문제 풀이 ]
```Java
import java.util.*;

class Solution {
    public int solution(int[] players, int m, int k) {
        int[] server = new int[24]; //시간별 서버 개수
        server[0] = 1;
        int addCnt = 0; //증설 횟수
        
        for (int time = 0; time < 24; time++) {
            if (time > 0) server[time] += server[time - 1];
            
            if (players[time] >= (server[time] * m)) {
                int totalCnt = (players[time] / m) + 1;
                int needCnt = totalCnt - server[time];
                addCnt += needCnt; //서버 증설
                if (time + k < 24) server[time + k] -= needCnt; //k 시간 후 반납
                server[time] = totalCnt;
            }
        }
        
        return addCnt;
    }

}
```
