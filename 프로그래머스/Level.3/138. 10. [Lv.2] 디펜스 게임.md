# 10. [Lv.2] 디펜스 게임

> **난이도**: 3
>
> **유형**: #구현


### [ 문제 풀이 ]
```Java
import java.util.*;
class Solution {
    public int solution(int n, int k, int[] enemy) {
        PriorityQueue<Integer> pq = new PriorityQueue<>(Collections.reverseOrder());
        int now = n;
        for (int i = 0; i < enemy.length; i++) {
            now -= enemy[i];
            pq.offer(enemy[i]);
            
            if (now < 0) { //남은 병사가 없을 경우, 가장 큰 라운드에 무적권 사용한걸로 처리
                if (k == 0) return i;
                now += pq.poll();
                k--;
            }
        }
        return enemy.length;
    }
}

```
