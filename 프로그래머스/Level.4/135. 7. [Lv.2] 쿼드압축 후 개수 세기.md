# 7. [Lv.2] 쿼드압축 후 개수 세기

### [ 문제 정보 ]
> **프로그래머스**: https://school.programmers.co.kr/learn/courses/30/lessons/68936?language=java
> 
> **난이도**: 4
>
> **유형**: #구현, #기타


### [ 문제 풀이 ]
```Java
import java.util.*;

class Solution {
    static int[] answer = new int[2];
    public int[] solution(int[][] arr) {
        compress(0, 0, arr.length, arr);
        return answer;
    }
    
    private void compress(int x, int y, int length, int[][] arr) {
        if (isSame(x, y, length, arr)) {
            answer[arr[x][y]]++;
            return;
        }
        int newLength = length / 2;
        compress(x, y, newLength, arr);
        compress(x, y+newLength, newLength, arr);
        compress(x+newLength, y, newLength, arr);
        compress(x+newLength, y+newLength, newLength, arr);
    }
    
    private boolean isSame(int x, int y, int length, int[][] arr) {
        //x, y 부터 시작해서 length 범위 탐색
        for (int i = x; i < x + length; i++) {
            for (int j = y; j < y + length; j++) {
                if (arr[x][y] != arr[i][j]) return false;
            }
        }
        return true;
    }
}
```
